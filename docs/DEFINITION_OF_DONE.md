## Per-Slice DoD

Each slice is DONE when:

1. All acceptance criteria (EARS) from BUILD_PLAN.md are met
2. Backend routes return correct responses for valid and invalid inputs
3. Frontend views render correctly and handle user interactions
4. No console errors in browser or backend terminal
5. Data persists across app restarts (SQLite)
6. New features don't break previously built slices
